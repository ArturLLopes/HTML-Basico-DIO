Tags HTML

No HTML, as tags desempenham um papel crucial para estruturar, semantizar e estilizar o conteúdo da página. Abaixo, explico como algumas tags importantes são usadas, com a perspectiva de um desenvolvedor front-end.

      Inicio Meio Fim

Tag h1 a h6:

São as tags de cabeçalho.

Elas organizam a hierarquia do conteúdo, com h1 sendo o mais importante (usado geralmente para o título principal da página) e h6 o menos importante. Sempre priorize o uso semântico correto para melhorar SEO e acessibilidade .

Tag p:

Define parágrafos

Define parágrafos de texto. É uma das tags mais usadas para conteúdo textual, trazendo semântica clara para o navegador e leitores de tela.

Tag mark:

Destaca texto

Destaca texto como se fosse marcado com um marcador fluorescente. É ótimo para atrair a atenção em contextos específicos.

Exemplo:

Este texto contém uma parte destacada.

Tag small:

Reduz o tamanho do texto

Small usado para textos menores. Pode ser usado para informações menos importantes, como notas de rodapé ou disclaimers.

Exemplo:

Este texto tem um rodapé em letras pequenas.

Tag i:

Aplica estilo em itálico

Originalmente usado para destaque visual, mas hoje seu uso é mais semântico, como para títulos de obras ou termosem outra língua.

Tag u:

Sublinha texto

Útil, mas deve ser usado com cuidado para não confundir o usuário, já que links geralmente também são sublinhados.

Exemplo:

Palavras em itálico e sublinhadas podem ser combinadas.

Tag strong:

Voltar ao inicio da pagina

Dá ênfase forte ao conteúdo

Geralmente exibido em negrito, e é semântico, indicando que o texto tem importância.

Exemplo:

Use negrito para destacar algo importante.

Tag ol , ul e li:

Criam listas

Ol é para listas ordenadas (numeradas) e ul para listas não ordenadas (com marcadores).
Li Define itens de lista, usados dentro de ol ou ul. Esses itens podem ser estilizados de várias formas com CSS.

Exemplo:
  1. Primeiro item
  2. Segundo item
  3. Terceiro item
  • Primeiro item
  • Segundo item
  • Terceiro item

Tag a:

Cria hiperlinks

Essenciais para navegação. Combine com o atributo href para definir o destino e torne os links acessíveis com descrições claras.

Exemplo:>

Visite o Example.com.

Tag hr:

Insere uma linha horizontal

Geralmente usada para separar seções de conteúdo. Ideal para criar divisores visuais.

Exemplo:

Tag sub e sup:

Posicionam texto em subscrito e sobrescrito

São úteis em fórmulas científicas ou referências .

Exemplo:

Fórmula química: H2O

Equação: E=mc2

Tag blockquote:

Voltar ao inicio da pagina

Destina-se a citações em bloco

Comumente estilizadas por navegadores para se destacarem visualmente.

Exemplo:
"O sucesso é a soma de pequenos esforços repetidos dia após dia." - Autor Desconhecido

Tag del:

Mostra texto como excluído

Com uma linha riscando-o. Útil para destacar alterações, como em sistemas de revisão.

Exemplo:

O preço antigo era $50, mas agora é $30.

Tag abbr:

Define abreviações ou acrônimos

Inclua o atributo title para fornecer o significado completo quando o usuário passar o mouse sobre o termo, melhorando a acessibilidade.

Exemplo:

O termo HTML é a base da web.

Tag font:

Permitia alterar fonte

Permitia alterar fonte, cor e tamanho, mas está obsoleta no HTML5. Substitua por CSS para estilizar texto. Porem obsoleto, não usar em novos projetos.

Exemplo:

Texto em azul e tamanho maior .
